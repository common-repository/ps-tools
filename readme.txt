PS Tools allows you insert a reference for any Adobe® Photoshop® tool in posts and pages.

== Description ==
By using PS Tools, you will not need to write up the same reference everytime, what is very boring and time consuming, you just need to select the tool and insert in your text. Also, if you deactivate the plugin, all images will show up, as long as you keep the plugin folder.
PS Tools do insert image and text in your content, no shortcodes!

Available languages:
*us-US (default)
*pt-BR


== Installation ==

1. Unzip and upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin.
1. Write your text as usual then click the PS Tool button where you want insert a tool reference.
1. Click the tool reference you want insert and click Insert.
1. The reference is inserted and depicts the tool button image and name.

== Frequently Asked Questions ==

= How to use reference in my language? =
If a translation file is available, you can use PS Tools referencing Photoshop® in your language, but please note: if you decide to use your language after sometime using this plugin, the old references will not be translated. The default language is en_US.
